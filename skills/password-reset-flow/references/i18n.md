# Password Reset Translations

## Passwords Namespace

```yaml
# config/locales/passwords.en.yml
en:
  passwords:
    new:
      title: "Forgot your password?"
      description: "Enter your email address and we'll send you a link to reset your password."
      back_to_login: "Back to sign in"
    edit:
      title: "Reset your password"
    form:
      email_address: "Email address"
      new_password: "New password"
      confirm_password: "Confirm new password"
      send_reset_link: "Send reset link"
      reset_password: "Reset password"
      placeholders:
        email: "Enter your email address"
        password: "Enter new password (min 8 characters)"
        confirm: "Confirm your new password"
    flash:
      create:
        success: "If an account exists with that email, you will receive password reset instructions shortly."
      update:
        success: "Your password has been reset successfully. Please sign in."
      invalid_token: "This password reset link is invalid or has expired. Please request a new one."
      rate_limit: "Too many password reset attempts. Please try again later."
    mailer:
      reset:
        subject: "Reset your password"
        heading: "Password Reset Request"
        intro: "We received a request to reset your password. Click the button below to choose a new password:"
        button: "Reset Password"
        expiry_notice: "This link will expire in 15 minutes."
        ignore_notice: "If you didn't request this, please ignore this email. Your password will remain unchanged."
        security_notice: "For security, this request was received from IP address %{ip}. If you didn't make this request, please contact support."
```

## Password Validation Errors

```yaml
# config/locales/en.yml
en:
  activerecord:
    errors:
      models:
        user:
          attributes:
            password:
              too_short: "must be at least %{count} characters"
              too_long: "must be at most %{count} characters"
              password_complexity: "must include at least one lowercase letter, one uppercase letter, and one number"
            password_confirmation:
              confirmation: "doesn't match password"
```
