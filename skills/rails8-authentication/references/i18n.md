# Authentication I18n Translations

```yaml
# config/locales/sessions.en.yml
en:
  sessions:
    new:
      title: "Sign in"
      forgot_password: "Forgot password?"
    form:
      email: "Email"
      password: "Password"
      sign_in: "Sign in"
      placeholders:
        email_address: "Enter your email address"
        password: "Enter your password"
    destroy:
      sign_out: "Sign out"
    flash:
      create:
        success: "You have been signed in."
        error: "Try another email address or password."
      destroy:
        success: "You have been signed out."
      rate_limit: "Try again later."
```

## Navigation Keys

```yaml
# config/locales/nav.en.yml
en:
  nav:
    settings: "Settings"
    sign_in: "Sign in"
    sign_out: "Sign out"
```

## Shared Keys

```yaml
# config/locales/shared.en.yml
en:
  shared:
    save: "Save"
    cancel: "Cancel"
    delete: "Delete"
    yes: "Yes"
    no: "No"
```

## Usage in Views

```erb
<!-- Always use I18n keys, never hardcode strings -->
<%= t('sessions.form.email') %>
<%= t('sessions.flash.create.success') %>
<%= link_to t('sessions.new.forgot_password'), new_password_path %>
```

## Usage in Controllers

```ruby
redirect_to root_path, notice: t("sessions.flash.create.success")
redirect_to new_session_path, alert: t("sessions.flash.create.error")
```
